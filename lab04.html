Conclusions

  1. Hemos hecho uso de JavaScript, por lo cuál hemos debido crear funciones que reacción al pulsar botones u opciones radio.
  2. En las funciones hemos guardado valores, podemos invocar estas desde un getElementById o un onclick.
  3. Se debe hacer de saber que el Script no necesariamente debe estar en el html como en este laboratorio, sino puede ser llamada desde otra carpeta.
  4. Se puede agregar CSS a los inputs que están siendo afectados por JavaScript ya que cada uno trabaja independientemente, además ayuda a que se estilice más nuestra página.
  5. Hemos hecho uso de inputs radio y text, cada uno realiza un call una vez se hace click sobre estos, así llamamos a las funciones script para que realicen la tarea.
